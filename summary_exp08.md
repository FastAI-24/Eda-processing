# Experiment 08: Spatial Clustering - 좌표 기반 공간 클러스터링

**날짜**: 2026-02-16
**목표**: 행정구역(동)의 경계를 넘어, 실제 좌표를 기반으로 한 '생활권' 클러스터를 생성하고 해당 지역의 가격 프리미엄을 모델이 정교하게 학습하도록 한다.

## 1. 방법론 (Methodology)

- **클러스터링**: 표준화된 좌표(위도, 경도) 데이터에 K-Means (K=150) 적용.
- **인코딩**: 데이터 누수(Leakage) 방지를 위해 K-Fold Target Encoding을 사용하여 `coord_cluster`별 평균 가격 산출.
- **계승**: Exp07의 품질 변수(`parking_per_unit` 등)와 최신 데이터 필터링 전략 유지.

## 2. 결과 (Results)

- **변수 중요도 (Feature Importance)**: `coord_cluster_mean_price`가 전체 변수 중 **압도적 1위** 차지!
- **해석**: 면적(`Area`)이나 단순 좌표보다, '어느 생활권에 속해 있는가'와 '그 생활권의 평균 단가가 얼마인가'가 가격 결정의 가장 핵심적인 요소임을 입증.
- **CV RMSE**: 공간 클러스터링 도입으로 지역적 특성을 더 세밀하게 포착하여 오차 감소.

## 3. 결론 (Conclusion)

- 공간 클러스터링은 서울의 비선형적인 입지 프리미엄을 함수화하는 데 매우 성공적이었음. 행정구역 기반의 모델링보다 훨씬 유연하고 강력한 성능을 보여줌.
