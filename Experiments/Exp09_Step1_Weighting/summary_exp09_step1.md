# Experiment 09-Step 1: All Data + Time Decay Weighting
**날짜**: 2026-02-16
**목표**: 금리 등 신규 변수 추가 전에 '전체 데이터 + 시간 가중치' 전략의 순수 효과를 증명한다.

## 1. 방법론 (Methodology)
- **데이터**: 2007~2023 전체 데이터 활용 (인근 좌표 복원 완료).
- **가중치**: `exp((year - 2023) * 0.1)` 적용 (2023년 대비 과거 데이터 비중 점진적 감소).
- **피처**: Exp08 수준 유지 (품질 변수 + 공간 클러스터링).
## 2. 결과 (Results)
- **CV RMSE**: **5,573** (전체 데이터 가중치 학습 기준).
- **상태**: 16,803점 실패 이후 '단계적 증명'을 위해 군더더기를 뺀 첫 번째 검증 작업.

## 3. 결론
- 리더보드 제출을 통해 과거 데이터가 가중치 제어 하에서 '힌트'가 되는지 아니면 '노이즈'가 되는지 판가름함.
