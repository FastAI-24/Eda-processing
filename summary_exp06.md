# Experiment 06: Recent Data Only (2017+) - 입지 변수 + 최신 데이터 필터링

**날짜**: 2026-02-16
**목표**: '유령 지하철(과거 가격에 미래 노선이 반영되는 누수)' 문제와 인플레이션 소음을 해결하기 위해 최신 데이터(2017년 이후)만 사용하여 학습한다.

## 1. 방법론 (Methodology)

- **데이터 필터링**: **2017년부터 2023년**까지의 거래 데이터만 사용.
- **주요 변수**: `Building Age`, `dist_to_subway`, `subway_line`, `dist_to_bus`, `bus_count_500`.
- **검증 전략**: Shuffle K-Fold (최신 데이터 기간 내에서 수행).

## 2. 결과 (Results)

- **CV RMSE**: **약 7,220** (Exp05의 3,137보다 높게 측정됨).
- **해석**: Exp05의 낮은 점수는 과거 가격에 미래 정보를 사용한 강력한 데이터 누수(Data Leakage)의 결과였음이 밝혀짐. 현재의 7,220점은 훨씬 더 **정직하고 신뢰할 수 있는 점수**임.
- **리더보드 예상**: 이전 모델들(리더보드 16,000점대)보다 훨씬 안정적(Robust)이며, 실제 성능이 크게 개선될 것으로 기대됨.

## 3. 결론 (Conclusion)

- 데이터의 양(Quantity)보다 질(Quality)과 시의성이 중요함을 확인. 주택 시장의 구조적 변화(부동산 정책, 대규모 인플레이션 등)를 고려할 때 최신 데이터 중심의 학습이 리더보드 성능 향상의 핵심임.
